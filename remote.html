ID : Aktivasi REMOTE PLAY tanpa login PSN


this simple video tutorial: YOUTUBE


1. Persiapkan terlebih dahulu perangkat untuk remote play (VITA/PC/Mobile)

2. aktivkan HEN dengan mode "OFFLINE" bisa pakai CACHE/DONGLE

3. logout ke menu konsol utama dan tekan (UP), pilih "Profil" -> "change user" (penting agar data save tidak hilang)

4. Mulai "SETTING" -> "NETWORK", HAPUS CENTANG "CONNECT TO INTERNET"

5. masuk ke "SETTING" -> "DEBUG SETTING" -> "SYSTEM" -> "MODE IDU", pilih "Aktif"

6. Klik "OK" untuk reboot.

7. Tunggu di sudut kanan atas ""Provide unit with a valid network connection or insert IDU update disc""

8. Tahan (R1) + (R2) + (L1) + (L2) + (Option) + (UP) dan tahan hingga Anda masuk ke Mode Staf

* Pastikan untuk menyalakan controller nya (direkomendasi sambungkan langsung ke console dengan usb)

9."SETTING" -> "REMOTE PLAY CONNECTION SETTING" -> "Direct Connection to PS Vita / PS TV System" -> "ADD DEVICE"

10. Masukkan kode yang muncul pada perangkat yang dijadikan remote play

* Pada proses ini, User2 akan muncul

* Ingat bahwa Anda hanya akan menggunakan koneksi wifi "tanpa internet". Menghubungkan melalui Internet tidak akan berfungsi

11. setelah berhasil tersambung, masuk kesetting - user guide - aktifkan HEN

12. Pergi "DEBUG SETTING" -> "SYSTEM -> "MODE IDU", pilih "Nonaktif" (proses ini mengembalikan ke NON-IDU mode)

13. Klik "OK" untuk reboot.




Jangan hapus USER2, jika tidak, REMOTE PLAY akan berhenti berfungsi!



EN : Activation REMOTE PLAY withoout PSN login


1. Prepare the first device to remote play (VITA / PC / Mobile)

2. activate HEN with "OFFLINE" mode can use CACHE / DONGLE

3. logout to main console menu and press (UP), choose "Profile" -> "change user" (add new user)

4. Start "SETTING" -> "NETWORK", REMOVE "CONNECT TO INTERNET"

5. Go to "SETTING" -> "DEBUG SETTING" -> "SYSTEM" -> "IDU MODE", select "On"

6. Click "OK" to reboot.

7. Wait in the upper right corner "" Provide units with valid network connections or insert IDU update disc ""

8. Hold (R1) + (R2) + (L1) + (L2) + (Option) + (UP) and hold it until you get into Staff Mode

* Be sure to its controller turn ON (recommended connect directly to the console with usb)

9. Go to "SETTING" -> "REMOTE PLAY CONNECTION SETTING" and select "Direct Connection to PS Vita / PS TV System" -> "ADD DEVICE"

10. Enter the code that appears on the device used for remote play

* In this process, User2 will appear

* Remember that you will only use wifi connection "without internet". Connecting over the Internet will not work

11. after successfully connected, enter setting - user guide - enable HEN

12. Go "DEBUG SETTING" -> "SYSTEM ->" IDU MODE ", select" Off "

13. Click "OK" to reboot.




Do not delete USER2, otherwise REMOTE PLAY will stop working!
